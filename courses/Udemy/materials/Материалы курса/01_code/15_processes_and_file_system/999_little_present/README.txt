Здесь две программы, которые используют системные утилиты, для отображения
в терминале показателей системы в забавном исполнении.
Значения именно команд для системных утилит немного расписал, чтоб было понятнее что
они делают и какой вывод. Конечно на разных ОС они разные.
Если что-то не работает, напишите, реализация довольно грубая(разбор строки вывода, чтоб забрать для отображения только нужное).
В этом практическом разделе в следующих лекциях мы будем работать с резервным копированием,
автоматизированным перемещением и другими операциями с файлами, поэтому это просто
забавные скрипты на которые будет интересно посмотреть и "поиграть" с ними.
Как пользоваться и где искать команды подобные тем, что я использовал в этих программах?:
- Ответ простой - гугл. Никто их все наизусть не знает, тем более в сложном исполнении.
- Конечно, знания о существовании этих утилит просто накапливаются по мере надобности,
  но все равно, за тонкостсями все равно в гугл.
В дальнейшем если понадобиться Вам работать с системными показателями ЦП, ОЗУ, данные
других датчиков, в python есть библитека (модуль) psutil, его нужно устанавливать дополнительно, мы это будем рассматривать отдельно.
