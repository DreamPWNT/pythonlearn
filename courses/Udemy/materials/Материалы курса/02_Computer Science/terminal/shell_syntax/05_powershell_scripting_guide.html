Это руководство поможет вам освоить базовый синтаксис PowerShell.

Эта информация не предназначена для заучивания, но поможет вам быстро сориентироваться, когда это понадобится.

1. Введение в PowerShell

PowerShell — это кроссплатформенная платформа автоматизации задач и управления конфигурацией, включающая командную оболочку и язык сценариев. Она используется для администрирования систем, автоматизации задач и управления конфигурацией в средах Windows, а также доступна на Linux и macOS.

2. Переменные

2.1 Объявление переменных

В PowerShell переменные объявляются с использованием символа $ перед именем переменной. Указывать тип данных не требуется, так как PowerShell использует динамическую типизацию. 


$name = "Alex"  # Строка
$age = 30       # Число (неявно но это int)
  • Переменные создаются в момент присваивания.
  • Пробелы вокруг = допустимы, но не рекомендуются для читаемости.
  • Значения по умолчанию интерпретируются как строки, если не указано иное.

2.2 Чтение значения переменной

Для доступа к значению переменной используйте её имя с префиксом $. 


echo $name  # Выводит: Alex

Рекомендуется использовать фигурные скобки ${} для явного указания границ имени переменной, особенно в строках: 


echo "${name}123"  # Выводит: Alex123

Без фигурных скобок PowerShell может интерпретировать $name123 как имя несуществующей переменной.

2.3 Изменение значения

Переменные можно переопределять, просто присваивая новое значение:


$name = "Sasha"
echo $name  # Выводит: Sasha

2.4 Типы переменных

PowerShell поддерживает различные типы данных, включая строки, числа, массивы и объекты. Тип переменной можно проверить с помощью метода GetType(). 


$name.GetType()  # Выводит: System.String
$age.GetType()   # Выводит: System.Int32

Для явного указания типа используйте квадратные скобки, например: 


[int]$age = 30

3. Переменные окружения

3.1 Доступ к переменным окружения

Переменные окружения доступны через виртуальный диск $env:.


$env:PATH  # Выводит переменную PATH

3.2 Установка переменных окружения

Для установки переменной окружения используйте $env:.


$env:MY_VAR = "value"

Пример проверки установленной переменной:


$env:MY_VAR  # Выводит: value

4. Арифметика

PowerShell поддерживает базовые арифметические операции с использованием операторов +, -, *, /. 


$count = 0
$count = $count + 1
echo $count  # Выводит: 1

Альтернативный синтаксис для инкремента:


$count++

Примечание: PowerShell автоматически выбирает подходящий тип данных для результата, например, System.Double для дробных чисел.

5. Условия и сравнения

5.1 Условные операторы If

Для выполнения кода на основе условий используйте конструкцию if. 


if ($name -eq "Alex") {
    echo "Привет, Alex!"
}

Важно: Заключайте переменные в кавычки в условиях, чтобы избежать ошибок при пустых значениях.

Пример с else:


if ($age -gt 18) {
    echo "Взрослый"
} else {
    echo "Несовершеннолетний"
}

5.2 Оператор Switch

Оператор switch позволяет выполнять различные блоки кода в зависимости от значения переменной. 


switch ($name) {
    "Alex" { echo "Привет, Alex!" }
    "Sasha" { echo "Привет, Sasha!" }
    default { echo "Привет, незнакомец!" }
}

5.3 Операторы сравнения

PowerShell предоставляет следующие операторы сравнения:

Оператор Описание
-eq Равно
-ne Не равно
-gt Больше
-lt Меньше
-ge Больше или равно
-le Меньше или равно

Пример:


if ($age -gt 18) {
    echo "Взрослый"
}

6. Циклы

6.1 Цикл For

Цикл for используется, когда известно количество итераций.


for ($i = 1; $i -le 5; $i++) {
    echo $i
}

6.2 Цикл Foreach

Цикл foreach удобен для перебора коллекций.


$array = 1, 2, 3, 4, 5
foreach ($item in $array) {
    echo $item
}

6.3 Цикл While

Цикл while выполняется, пока условие истинно.


$i = 1
while ($i -le 5) {
    echo $i
    $i++
}

6.4 Цикл Do-While

Цикл do-while выполняется хотя бы один раз, затем продолжается, пока условие истинно. 


$i = 1
do {
    echo $i
    $i++
} while ($i -le 5)

7. Функции

7.1 Определение функций

Функции определяются с помощью ключевого слова function.


function Greet {
    param (
        [string]$name
    )
    echo "Привет, $name!"
}

Greet -name "Alex"  # Выводит: Привет, Alex!

7.2 Параметры

Функции могут принимать параметры, которые могут иметь значения по умолчанию. 


function Greet {
    param (
        [string]$name = "Guest"
    )
    echo "Привет, $name!"
}

Greet  # Выводит: Привет, Guest!

8. Массивы

8.1 Создание массивов

Массивы создаются путём разделения элементов запятыми.


$array = 1, 2, 3, 4, 5

8.2 Доступ к элементам массива

Элементы массива доступны по индексам, начиная с 0.


echo $array[0]  # Выводит: 1

8.3 Изменение массивов

Массивы можно изменять, присваивая новые значения по индексам.


$array[0] = 10
echo $array[0]  # Выводит: 10

9. Работа с файлами

9.1 Чтение файлов

Используйте Get-Content для чтения содержимого файла.


$content = Get-Content -Path "file.txt"
echo $content

9.2 Запись в файлы

Используйте Set-Content для записи в файл.


Set-Content -Path "file.txt" -Value "Привет, мир!"

9.3 Добавление в файлы

Используйте Add-Content для добавления текста в файл.


Add-Content -Path "file.txt" -Value "Это добавленный текст."

10. Аргументы командной строки

10.1 Доступ к аргументам

Аргументы командной строки доступны через $args или через определение параметров. 


param (
    [string]$name
)

echo "Привет, $name!"

Запустите скрипт с помощью: .\script.ps1 -name "Alex"

11. Ввод пользователя

11.1 Запрос ввода

Используйте Read-Host для запроса ввода у пользователя.


$name = Read-Host "Введите ваше имя"
echo "Привет, $name!"

12. Значения по умолчанию

12.1 Установка значений по умолчанию для параметров

Значения по умолчанию можно задавать для параметров функций.


function Greet {
    param (
        [string]$name = "Guest"
    )
    echo "Привет, $name!"
}

Greet  # Выводит: Привет, Guest!

12.2 Использование $PSDefaultParameterValues

Глобальные значения по умолчанию для параметров командлетов можно задавать с помощью $PSDefaultParameterValues. 


$PSDefaultParameterValues = @{
    "Send-MailMessage:SmtpServer" = "smtp.example.com"
}

13. Примеры

13.1 Простой счётчик


$count = 0
while ($count -lt 5) {
    echo "Счётчик: $count"
    $count++
}

13.2 Проверка существования файла


if (Test-Path "file.txt") {
    echo "file.txt существует"
} else {
    echo "file.txt не существует"
}

13.3 Цикл по файлам


foreach ($file in Get-ChildItem *.txt) {
    echo "Обработка файла: $file"
}

14. Подсказки и рекомендации

  • Заключайте переменные в кавычки: Используйте "$var" в условиях, чтобы избежать ошибок с пустыми значениями.
  • Используйте Write-Output вместо echo в файлах сценариях: Это более формальный способ вывода данных.
  • Избегайте бесконечных циклов: Убедитесь, что условие цикла рано или поздно станет ложным.
  • Документируйте функции: Используйте комментарии для описания параметров и назначения функций.
  • Проверяйте ввод: Используйте атрибуты валидации, такие как [ValidateSet], для параметров функций.
