# Специальные переменные cmd:

| Переменная        | Что возвращает                             |
| ----------------- | ------------------------------------------ |
| `%CD%`            | Текущий каталог                            |
| `%DATE%`          | Текущая дата                               |
| `%TIME%`          | Текущее время                              |
| `%RANDOM%`        | Случайное число от 0 до 32767              |
| `%ERRORLEVEL%`    | Код возврата последней команды             |
| `%CMDEXTVERSION%` | Версия расширений командного процессора    |
| `%CMDCMDLINE%`    | Строка запуска текущего `cmd.exe` процесса |
| `%USERNAME%`      | Имя текущего пользователя                  |


### Таблица спецсимволов prompt
| Код  | Значение                                 |                         |
| ---- | ---------------------------------------- | ----------------------- |
| `$P` | Текущий путь (например, `C:\Users\Alex`) |                         |
| `$G` | `>` (стрелка)                            |                         |
| `$L` | `<`                                      |                         |
| `$Q` | `=`                                      |                         |
| `$B` | \`                                       | \` (вертикальная черта) |
| `$T` | Текущее время                            |                         |
| `$D` | Текущая дата                             |                         |
| `$N` | Диск (например, `C`)                     |                         |
| `$V` | Версия командного процессора             |                         |
| `$S` | Пробел                                   |                         |
| `$$` | Буквально символ `$`                     |                         |

# Специальные переменные sh/bash/zsh:

### Переменная `PS1` и настройка приглашения в bash/zsh

Переменная `PS1` определяет, как выглядит командная строка (prompt).

Примеры:

```bash
PS1='\u@\h:\w\$ '
```

| Элемент | Что показывает                      |
| ------- | ----------------------------------- |
| `\u`    | Имя пользователя                    |
| `\h`    | Имя хоста (до первой точки)         |
| `\H`    | Полное имя хоста                    |
| `\w`    | Текущий путь (относительно `$HOME`) |
| `\W`    | Имя текущего каталога               |
| `\t`    | Текущее время (HH\:MM\:SS)          |
| `\T`    | Текущее время (12-часовой формат)   |
| `\d`    | Дата (день недели, месяц, день)     |
| `\!`    | Номер текущей команды в истории     |
| `\#`    | Номер команды с начала сессии       |
| `\$`    | `#` если root, иначе `$`            |
| `\n`    | Перенос строки                      |
| `\\`    | Символ `\`                          |

Другие важные переменные:

* `PS2` — приглашение для продолжения многострочной команды (по умолчанию `>`).
* `PS4` — префикс при трассировке (`set -x`), по умолчанию `+`.

---

## Пример красивого `PS1`:

```bash
PS1='\[\e[1;32m\]\u@\h \[\e[1;34m\]\w\[\e[0m\]\n\$ '
```
\e - это то самое \033 как в синтаксисе python.

Цветной prompt:

* `\[\e[1;32m\]` — ярко(bold)-зелёный
* `\[\e[1;34m\]` — ярко(bold)-синий
* `\[\e[0m\]` — сброс цвета
