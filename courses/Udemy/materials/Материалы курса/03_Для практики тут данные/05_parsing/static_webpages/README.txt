Чтоб "запустить" этот сайт через мини сервер, в этой директории нужно
открыть терминал и:
python(3)(3.13) -m http.server 8000
Сайт будет доступен по адресу http://localhost:8000 (локально только на вашем ПК).
8000 это порт. На одном адресе могут работать множество разных программ и они
различаются по порту. Как дом с квартирами.


Если просто двумя кликами открыть файл index.html, то он просто
откроется в браузере и браузер будет перемещатся по ссылкам по файловой
системе. Так уж это работает, потому что ссылки в сайте указаны в контексте
файловой системы. Но в таком режиме нельзя получить доступ к страницам сайта
програмно (через python например) по localhost и порт.
